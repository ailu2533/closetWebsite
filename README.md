# 柠檬衣橱官方网站

这是柠檬衣橱app的静态官方网站，采用模块化设计，优化了移动端体验。

## 🌐 在线访问

网站已部署到GitHub Pages，可通过以下链接访问：

**官方网站：** https://ailu2533.github.io/closetWebsite

## 🚀 快速开始

### 本地开发
```bash
# 克隆仓库
git clone https://github.com/ailu2533/closetWebsite.git

# 进入项目目录
cd closetWebsite

# 启动本地服务器
npm run dev
# 或者
python -m http.server 8000

# 在浏览器中访问 http://localhost:8000
```

### 自动部署
项目配置了GitHub Actions自动部署，每次推送到main分支时会自动部署到GitHub Pages。

## 项目结构

```
closetWebsite/
├── index.html              # 主页面文件
├── android-migration.html  # 安卓版数据迁移指南页面
├── index-backup.html       # 原始文件备份
├── css/                    # 样式文件目录
│   ├── styles.css          # 主要样式文件
│   └── responsive.css      # 响应式样式文件
├── js/                     # JavaScript文件目录
│   └── main.js             # 主要功能脚本
├── package.json            # 项目配置文件
└── README.md               # 项目说明文件
```

## 功能特性

### 🎨 设计特色
- **柠檬主题**：采用柠檬黄到橙色的渐变配色
- **响应式设计**：完美适配桌面、平板和手机设备
- **动画效果**：流畅的CSS3动画和交互效果
- **现代化UI**：简洁美观的界面设计

### 📱 移动端优化
- **安卓优先**：在移动端突出显示安卓下载按钮
- **智能适配**：根据屏幕尺寸自动调整布局
- **触摸友好**：优化了移动设备的触摸体验
- **性能优化**：减少了不必要的资源加载

### 📊 数据迁移功能
- **柚子衣橱数据迁移**：为原柚子衣橱用户提供完整的数据迁移方案
- **四步迁移流程**：清晰的步骤指导，确保数据无缝转移
- **醒目迁移入口**：专门的数据修复版下载区域
- **用户友好提示**：详细的操作说明和温馨提示

### 🔧 技术特性
- **模块化架构**：CSS和JavaScript分离，便于维护
- **SEO优化**：完整的meta标签和语义化HTML
- **兼容性好**：支持现代浏览器和移动设备
- **无依赖**：纯HTML/CSS/JavaScript，无需框架

## 开发说明

### 文件说明

#### CSS文件
- `styles.css`：包含所有基础样式和组件样式
- `responsive.css`：包含所有响应式媒体查询

#### JavaScript文件
- `main.js`：包含所有交互功能和初始化代码

#### HTML文件
- `index.html`：主页面，引用外部CSS和JS文件
- `android-migration.html`：安卓版数据迁移详细指南页面
- `index-backup.html`：原始单文件版本的备份

### 修改指南

1. **样式修改**：编辑 `css/styles.css` 或 `css/responsive.css`
2. **功能修改**：编辑 `js/main.js`
3. **内容修改**：编辑 `index.html`

### 部署说明

这是一个纯静态网站，可以直接部署到任何静态托管服务：

- **GitHub Pages** ✅ (当前使用)
- Netlify
- Vercel
- 阿里云OSS
- 腾讯云COS

#### GitHub Pages 部署步骤
1. 将代码推送到GitHub仓库的main分支
2. GitHub Actions会自动构建和部署
3. 访问 https://ailu2533.github.io/closetWebsite

#### 手动部署到其他平台
只需上传所有文件到服务器根目录即可。
